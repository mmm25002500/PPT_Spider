# PTT爬蟲 PTT-Spider

這是一款可以將PTT所有資訊擷取下來的程式，可隨時獲得PTT版標題且保存至本地

## 安裝 Install

Windows 10:

1.下載Python

2.開啟cmd

``cd PTT_Spider``

``python main.py``



Linux(Ubuntu):

``sudo apt update``

``sudo apt install python python3 python2 git ``

``cd  PTT_Spider``

``python main.py``
